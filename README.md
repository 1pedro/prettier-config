# @1pedro/prettier-config


## Usage

**Install**:

```sh
$ npm install -D @1pedro/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@1pedro/prettier-config"
}
```

[Read more on sharing configurations.](https://prettier.io/docs/en/configuration.html#sharing-configurations)
